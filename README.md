# KeenKeeper

KeenKeeper is a personal relationship management app that helps you stay connected with the people who matter most. Track your friendships, log interactions, and never lose touch again.


## 🛠️ Technologies Used

- **React.js** — Frontend UI
- **React Router DOM** — Page navigation
- **Tailwind CSS** — Styling and layout
- **DaisyUI** — UI component library
- **Recharts** — Friendship analytics chart
- **React Hot Toast** — Toast notifications
- **Vite** — Build tool

## ✨ Features

- 📋 View all your friends in a clean card grid with status indicators
- 👤 Detailed friend profile page with contact stats and relationship goals
- ⚡ Quick Check-In buttons (Call, Text, Video) that log interactions instantly
- 📜 Timeline page showing full interaction history with filter options
- 📊 Friendship Analytics page with a pie chart of interaction types
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🔔 Toast notifications on every check-in action
- 🚫 Custom 404 page for unknown routes
